## 📜 설명
- React/Vite를 통해 만든 교통약자분들을 위한 정보 사이트 입니다.
- 지방자치단체의 교통약자이동지원센터에 관한 정보와 역사 내 시설 여부를 확인하실 수 있습니다.

## ⏰기간
- 2025.02.03 -2025.02.08 (6일)


## ⚙️ 기능
- 📚 교통약자이동지원센터 정보 카카오맵으로 보기
- 📖 역사 내 시설 검색 기능

## 🛠️ 기술 스택
- **프레임워크**: React
- **스타일**: Tailwind CSS
- **언어**: TypeScript, JavaScript, HTML, CSS
- **언어**:vite (빌드 도구 & 개발 서버)
